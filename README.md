# Nyota翻翻乐游戏 - 独立版本

## 📦 文件说明

这是一个完全独立的Nyota翻翻乐H5游戏包，可以直接在任何Web服务器上部署使用。

### 📁 文件结构

```
nyota-game-standalone/
├── index.html          # 主HTML文件（包含所有游戏逻辑）
├── README.md           # 本说明文件
└── assets/             # 资源文件夹
    ├── nyota logo.png                           # Nyota Logo
    ├── 放松 露营 周末 轻快 旅行 vlog日常 做饭 治愈_爱给网_aigei_com.mp3  # 背景音乐
    └── *.png                                    # 10张Nyota游戏卡牌图片
```

## 🚀 使用方法

### 方法1：本地直接打开
直接双击 `index.html` 文件，在浏览器中打开即可使用。

### 方法2：本地服务器（推荐）
使用任何本地Web服务器运行，例如：

```bash
# 使用Python
cd nyota-game-standalone
python -m http.server 8000

# 然后在浏览器访问
# http://localhost:8000
```

### 方法3：部署到服务器
将整个文件夹上传到任何Web服务器即可：
- Nginx/Apache
- Vercel/Netlify
- GitHub Pages
- 任何静态网站托管服务

## 🎮 游戏功能

### 1. 云朵主题开场页
- 动态云朵背景
- 温柔和缓的入场动画
- 点击"开始慢游"进入游戏

### 2. Nyota慢游场景
- **晨间治愈**：用一杯咖啡的时间，和自己和解
- **午后放空**：让思绪随蒲公英一起飘远
- **夜晚疗愈**：在星空下，找回内心的宁静

### 3. 慢游收藏家
- 点击场景卡片收集拼图碎片
- 集齐3块碎片完成拼图

### 4. Nyota翻翻乐
- 20张卡牌（10种类型×2张）
- 配对游戏机制
- 可爱的庆祝动画
- 背景音乐播放

### 5. 品牌展示区
- 展示3款Nyota手办
- 点击"立即购买"跳转到微信小程序
- 小程序链接：`weixin://dl/business/?t=Ltcq1cyK7kb`

## 📱 移动端适配

- 完美适配各种屏幕尺寸
- 响应式布局设计
- 支持触屏操作

## 🔧 自定义修改

### 修改微信小程序链接
找到文件中的 `goToStore()` 函数（约第1180行），修改以下链接：

```javascript
const miniProgramUrl = 'weixin://dl/business/?t=YOUR_SCHEME_ID';
```

### 修改背景音乐
1. 替换 `assets/` 文件夹中的音频文件
2. 确保文件名保持一致，或修改代码中的文件名引用

### 修改游戏图片
- 替换 `assets/` 文件夹中的卡牌图片
- 图片尺寸建议：200×200像素或保持原有比例

## 💡 注意事项

1. **音频播放**：移动端浏览器要求用户交互后才能播放音频，首次播放需点击"播放音乐"按钮
2. **外部图片**：部分图片使用了外部链接（场景卡片、品牌手办等），这些链接会从网络加载
3. **浏览器兼容性**：支持所有现代浏览器（Chrome、Safari、Firefox、Edge等）
